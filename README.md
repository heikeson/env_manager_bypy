# 环境变量管理器

一个适用于Windows操作系统的环境变量管理工具，使用Python和wxPython开发。

## 功能特性

- **环境变量管理**：查看、添加、编辑、删除系统级和用户级环境变量
- **导入/导出功能**：支持导出环境变量到JSON和文本文件，以及从这些文件导入
- **安全特性**：系统级环境变量操作需要管理员权限，对系统关键环境变量进行特殊保护
- **直观界面**：使用wxPython构建友好的图形界面
- **错误处理**：完善的错误处理机制
- **日志记录**：详细的操作日志记录

## 系统要求

- 操作系统：Windows 10 及以上版本
- Python 版本：3.8 及以上
- 依赖包：wxPython 4.2.2 及以上

## 安装方法

1. 克隆仓库：
   ```bash
   git clone <repository-url>
   cd 环境变量管理器
   ```

2. 安装依赖：
   ```bash
   pip install wxPython
   ```

## 使用方法

1. 运行应用程序：
   ```bash
   python main.py
   ```

2. 基本操作：
   - **查看环境变量**：在下拉菜单中选择作用域（用户/系统）
   - **添加环境变量**：点击"添加"按钮，输入变量名和值
   - **编辑环境变量**：选择要编辑的变量，点击"编辑"按钮
   - **删除环境变量**：选择要删除的变量，点击"删除"按钮
   - **导出环境变量**：点击"导出"按钮，选择保存位置和文件类型
   - **导入环境变量**：点击"导入"按钮，选择要导入的文件

## 注意事项

- 修改系统级环境变量需要管理员权限
- 系统关键环境变量（如PATH、SYSTEMROOT等）修改时会显示警告，请谨慎操作
- 环境变量修改后，某些应用程序可能需要重启才能看到变化

## 故障排除

如果遇到问题，可以查看logs目录下的日志文件获取详细信息。常见的错误包括：

- 权限不足：需要以管理员权限运行
- 文件操作错误：检查文件路径和权限
- 环境变量操作错误：检查环境变量名是否合法
