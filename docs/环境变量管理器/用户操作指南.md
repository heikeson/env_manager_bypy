# 环境变量管理器用户操作指南

## 1. 应用程序简介

环境变量管理器是一个适用于Windows操作系统的工具，用于管理系统级和用户级环境变量。它提供了直观的图形界面，方便用户查看、添加、编辑和删除环境变量，同时支持环境变量的导入和导出功能。

### 主要功能
- 查看系统级和用户级环境变量
- 添加新的环境变量
- 编辑现有环境变量
- 删除环境变量
- 导出环境变量到JSON或文本文件
- 从JSON或文本文件导入环境变量
- 环境变量变更的即时预览和确认
- 权限验证和安全检查

## 2. 系统要求

- 操作系统：Windows 10 及以上版本
- Python 版本：3.8 及以上
- 依赖包：wxPython 4.2.2 及以上

## 3. 安装和运行

### 3.1 安装依赖

在运行应用程序之前，需要安装wxPython库：

```bash
pip install wxPython
```

### 3.2 运行应用程序

1. 打开命令提示符或PowerShell
2. 导航到应用程序所在目录
3. 执行以下命令：

```bash
python main.py
```

## 4. 操作指南

### 4.1 查看环境变量

1. 启动应用程序后，默认显示用户级环境变量
2. 在"作用域"下拉菜单中选择"系统"可以查看系统级环境变量
3. 环境变量列表显示变量名和对应的值

### 4.2 添加环境变量

1. 点击"添加"按钮
2. 在弹出的对话框中输入环境变量名和值
3. 点击"确定"按钮
4. 系统会提示确认操作，点击"是"继续
5. 如果添加成功，会显示成功提示

### 4.3 编辑环境变量

1. 在环境变量列表中选择要编辑的变量
2. 点击"编辑"按钮
3. 在弹出的对话框中修改环境变量名和/或值
4. 点击"确定"按钮
5. 系统会提示确认操作，点击"是"继续
6. 如果修改成功，会显示成功提示

### 4.4 删除环境变量

1. 在环境变量列表中选择要删除的变量
2. 点击"删除"按钮
3. 系统会提示确认操作，点击"是"继续
4. 如果删除成功，会显示成功提示

### 4.5 导出环境变量

1. 选择要导出的环境变量作用域（用户或系统）
2. 点击"导出"按钮
3. 在弹出的文件选择对话框中选择保存位置和文件类型（JSON或文本）
4. 点击"保存"按钮
5. 如果导出成功，会显示成功提示

### 4.6 导入环境变量

1. 选择要导入到的环境变量作用域（用户或系统）
2. 点击"导入"按钮
3. 在弹出的文件选择对话框中选择要导入的文件
4. 点击"打开"按钮
5. 系统会询问是否覆盖已存在的环境变量，根据需要选择
6. 如果导入成功，会显示成功提示

## 5. 注意事项

### 5.1 权限要求

- 修改系统级环境变量需要管理员权限
- 如果当前没有管理员权限，系统会提示并尝试获取管理员权限

### 5.2 安全警告

- 系统关键环境变量（如PATH、SYSTEMROOT等）修改时会显示警告
- 修改这些变量可能影响系统稳定性，请谨慎操作

### 5.3 操作日志

- 应用程序会记录所有操作到logs目录下的env_manager.log文件
- 日志文件会自动轮转，最多保留5个备份文件

### 5.4 环境变量生效

- 修改环境变量后，系统会自动通知其他应用程序
- 某些应用程序可能需要重启才能看到环境变量的变化

## 6. 常见问题

### 6.1 无法修改系统环境变量

- 确保以管理员权限运行应用程序
- 检查是否有其他程序正在使用该环境变量

### 6.2 导入环境变量失败

- 检查导入文件格式是否正确
- 确保文件路径正确且有读写权限
- 对于系统级环境变量，确保有管理员权限

### 6.3 环境变量不生效

- 重启相关应用程序
- 注销并重新登录Windows
- 重启计算机

## 7. 故障排除

如果遇到问题，可以查看logs目录下的日志文件获取详细信息。常见的错误包括：

- 权限不足：需要以管理员权限运行
- 文件操作错误：检查文件路径和权限
- 环境变量操作错误：检查环境变量名是否合法
